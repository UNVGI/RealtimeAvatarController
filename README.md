# RealtimeAvatarController

VTuber 向けのリアルタイムモーション受信・アバター切替ツールです。Slot ベースの MoCap ソース管理、アバタープロバイダ抽象化、モーションパイプラインを Unity 向けに提供します。

## パッケージ

本体は UPM パッケージとして `RealtimeAvatarController/Packages/com.hidano.realtimeavatarcontroller/` に格納されています。導入手順と利用方法は以下を参照してください。
VMC 受信は別パッケージ `RealtimeAvatarController/Packages/com.hidano.realtimeavatarcontroller.mocap-vmc/` で提供し、利用者準備は uOSC のみ依存です。
