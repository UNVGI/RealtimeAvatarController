# ui-sample T11-2 実装ログ

## タスク
SlotManagerBehaviour の initialSlots への参照共有用アセット設定

## 対象
Samples~/UI/Scenes/SlotManagementDemo.unity 内の [SlotManager] GameObject にアタッチされている SlotManagerBehaviour コンポーネントの `initialSlots` 配列。

## 実施内容
- [SlotManager] GameObject の SlotManagerBehaviour (MonoBehaviour fileID 100100002) の `initialSlots` に、T11-1 で作成した以下 2 アセットを追加:
  - SlotSettings_Shared_Slot1.asset (guid: a07b8c9d0e1f2a3b4c5d6e7f8091a2b3)
  - SlotSettings_Shared_Slot2.asset (guid: b18c9d0e1f2a3b4c5d6e7f8091a2b3c4)
- 両アセットは同一の VMCMoCapSourceConfig_Shared.asset を参照しており、Play Mode 起動時に SlotManager.AddSlotAsync() が両スロットに対し実行されることで IMoCapSourceRegistry の参照共有 (refCount=2) が成立する想定。

- [x] 各要素は type: 2 (asset) としてアセット参照を示している
